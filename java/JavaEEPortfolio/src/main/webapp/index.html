My Java EE Projects

This portfolio displays the Java EE web projects I make for my school. This portfolio itself is homework.

The design of this page is directly inspired by bettermotherfuckingwebsite.com.

  • Rock Paper Scissors

    A simple Rock Paper Scissors game played against the computer. A Servlet is used to redirect the player toward JSP pages, protected in the WEB-INF directory.

  • Guess the Number

    Guess the Number is a simple app that generates a random number and lets the user guess what it is. A Servlet is used to redirect the user to a "Win" or "Fail" page depending on the number he sent through a form. The administrator can edit the random number's min and max values in the Servlet's init parameters.
